# 摘要

随着汽车制造行业的快速发展，企业面临着生产计划排程效率低、无法快速响应市场需求变化等问题。高级计划排程系统（APS）作为连接企业资源计划（ERP）与制造执行系统（MES）的桥梁，能够在复杂约束条件下生成优化的生产计划，是解决上述问题的有效途径。本课题以汽车制造企业为应用背景，设计并实现了一套基于B/S架构的智能排产系统。系统后端采用Spring Boot框架构建，前端使用Vue.js框架开发，数据库采用MySQL并结合Redis进行缓存优化。在排产优化方面，系统基于OptaPlanner约束求解框架实现了智能排产求解模块，通过定义硬约束（资源匹配、时间无重叠、车间工序顺序）和软约束（优先级排序、最小化完工时间、最小化任务间隙），采用元启发式搜索算法在考虑物料约束、工艺约束、资源能力约束等多重约束条件下，自动生成最优或近优的生产计划和调度方案。经测试验证，本系统能够有效完成汽车制造场景下的生产排产任务，排产结果能够明显减少生产等待时间、提高设备利用率，具有一定的实用价值。

**关键词**：智能排产；APS；汽车制造；约束求解；OptaPlanner

# Abstract

With the rapid development of the automobile manufacturing industry, enterprises are facing challenges such as low efficiency in production planning and scheduling, and inability to quickly respond to market demand changes. Advanced Planning and Scheduling (APS), as a bridge between Enterprise Resource Planning (ERP) and Manufacturing Execution System (MES), can generate optimized production plans under complex constraint conditions, which is an effective solution to the above problems. This project takes the automobile manufacturing enterprise as the application background, and designs and implements an intelligent production scheduling system based on B/S architecture. The backend adopts Spring Boot framework, the frontend uses Vue.js framework, and the database uses MySQL combined with Redis for cache optimization. In terms of scheduling optimization, the system implements an intelligent scheduling solver based on OptaPlanner constraint solving framework, which defines hard constraints (resource matching, no time overlapping, workshop sequence) and soft constraints (priority scheduling, minimizing makespan, minimizing task gaps), and employs metaheuristic search algorithms to automatically generate optimal or near-optimal production plans and scheduling schemes under multiple constraints such as material constraints, process constraints, and resource capacity constraints. Through testing and verification, the system can effectively complete production scheduling tasks in automobile manufacturing scenarios, and the scheduling results can significantly reduce production waiting time and improve equipment utilization rate, which has certain practical value.

**Keywords**: Intelligent Production Scheduling; APS; Automobile Manufacturing; Constraint Solving; OptaPlanner
